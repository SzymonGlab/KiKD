244755 Szymon Głąb

Program został napisany w Pythonie w wersji 3.7.4. Wszystkie kody źródłowe znajdują się w pliku l2.py.

Program implemenruje dynamiczne kodowanie Huffmana. Alfabetem wejściowym dla programu są 8-bitowe kody ASCII.

Program można uruchomić w dwóch opcjach:

    1. kodowanie ------> python3 l2.py code <plik wejściowy> <plik wyjściowy>
    2. dekodowanie ----> python3 l2.py decode <plik wejściowy> <plik wyjściowy>
